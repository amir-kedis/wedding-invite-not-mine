This should be a wedding invitation website based on https://sample.weddingg.co/ .

## Technical Requirements

- Should be made with nextjs
- Should be hosted using vercel in the command line
- Should support internationlization (Arabic by default and english)
- Should be easy to configure (change all the text, images, videos, disable and enabled sections etc)
- If a database is needed, it should be simple and easy to use (firebase, supabase is a good option) it should be free
- It should be responsive and mobile first (most mobile users)
- Assets are on ./data_acuisition
- It should be clean and well codded and secure.
- Use tailwind
- Use framer-motion for animations
- For commits follow conventional commits
- Make a good .gitignore and follow best standards on whay should be in the repo.

## Functinoal Requirements

- It should have all the section that are in the sample website
- It should start with the same letter video and image that when clicked it shows the envelope opening and then the site.
- It should have scroll animations and interactivites
